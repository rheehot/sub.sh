Linux settings of Heungsub Lee

sub.sh is a shell script to terraform a Linux environment as for my taste. The script aims ZSH on Ubuntu what I feel comfortable. It also configures Vim and Python environments to maximize my productivity.

Take my settings for your Linux environment:

$ curl -sL sub.sh | bash

Or customize by options:

$ curl -sL sub.sh | bash -s - --help
